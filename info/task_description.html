Remove accents

Assuming you are developing a member system like facebook and you want to provide a functionality to search other members regardless of accents in username without using 3rd party collation library, you will need to remove accents from username before comparing them.

Input: Unicode string

Output: Unicode string

Example: 

checkio(u"préfèrent") == u"preferent"
checkio(u"loài trăn lớn") == u"loai tran lon"
